# Jenkins Fundamentals

`Jenkins` is a powerful open-source automation server that helps automate various aspects of software development, including building, testing, and deploying applications.

In this folder you will find resources to learn the basics of `Jenkins`, including how to set up a Jenkins server, create and manage Jenkins pipelines, and integrate Jenkins with other tools in your development workflow.
