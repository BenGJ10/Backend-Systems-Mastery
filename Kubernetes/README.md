# Kubernetes Fundamentals

`Kubernetes` is an open-source container orchestration platform that automates the deployment, scaling, and management of containerized applications.

In this folder you will find resources to learn the basics of `Kubernetes`, including how to set up a Kubernetes cluster, deploy applications, and manage your Kubernetes environment effectively.

Understanding Kubernetes is essential for modern software development and operations, as it provides a powerful framework for running applications in a scalable and resilient manner. So take your time to explore the resources in this folder and get hands-on experience with `Kubernetes`!
